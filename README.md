Las instrucciones para ejecutar este codigo son:

-Abrir los archivos index.html y prueba.js con un IDE. (Lo hice con Visual Studio Code)
-Abrir el archivo index.html en el navegador. (Se puede abrir manualmente dando doble clic o al usar VSCode dando clic derecho en el archivo 'index.html' y seleccionando 'Open with Five Server')
-Se pueden cambiar los inputs en cada desafio mediante la asignacion del nuevo valor a las siguientes variables.
    PRIMER DESAFIO: Variable 'x'.(Que es el numero que se pretende evaluar si es palindromo)
    SEGUNDO DESAFIO: Arreglo 'numeros' y variable 'buscar'.('numeros' es el arreglo y 'buscar' es la variable en la que se define que numero se pretende encontrar)
    TERCER DESAFIO: Variable 's'.(Que es la variable en la que se define el texto el cual se dividira en palabras)
-Los resultados, inputs, arreglos se mostraran en el navegador mediante el archivo index.html, tambien puede verse dando clic derecho en la pagina, dando clic en 'inspeccionar elemento' e ingresando a la pesta√±a 'console', o en la terminal del IDE VSCode.
